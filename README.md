# Tagpo Landing Page

Official landing page for Tagpo.

## Files

- `index.html` - the website entry point
- `assets/` - logos, app screenshots, and school marks
- `.nojekyll` - keeps GitHub Pages from processing the static site through Jekyll

## Publish With GitHub Pages

1. Create a new GitHub repository.
2. Upload everything in this folder to the root of the repository.
3. Go to the repository's `Settings` tab.
4. Open `Pages`.
5. Under `Build and deployment`, choose `Deploy from a branch`.
6. Set branch to `main` and folder to `/(root)`.
7. Save. GitHub will give you the live website URL after it finishes deploying.

The site is static, so it does not need `server.js`, npm, or a backend to publish.
